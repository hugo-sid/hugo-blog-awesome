Go 언어로 작성된 Hugo는 [Apache Licence 2.0.](https://github.com/gohugoio/hugo/blob/master/LICENSE)하에 이용 가능한 정적 사이트 생성입니다. Hugo는 TOML, YAML 그리고 JSON 데이터 파일 종류, 마크다운과 HTML 컨텐츠 파일을 지원하고 숏코드를 이용해 리치 컨텐츠를 추가합니다. 다른 주목할만한 기능으로 분류 체계, 다국어 모드, 이미지 처리, 사용자 지정 출력 형식, HTML/CSS/JS 최소화와 Sass SCSS 작업 흐름이 있습니다.

Hugo는 여러가지 오픈소스 프로젝트를 이용하며 이는 다음을 포함합니다.

* https://github.com/yuin/goldmark
* https://github.com/alecthomas/chroma
* https://github.com/muesli/smartcrop
* https://github.com/spf13/cobra
* https://github.com/spf13/viper

Hugo는 블로그, 회사 웹사이트, 창의적인 포트폴리오, 온라인 잡지, 단일 페이지 앱 그리고 심지어는 수 천개의 페이지를 가지는 웹사이트에 적합합니다.

Hugo는 복잡한 런타임, 의존성, 그리고 데이터베이스를 설정해야하는 부담 없이 자신의 웹사이트를 직접 만들고 싶은 사람들을 위해 만들어졌습니다.

Hugo로 만들어진 웹사이트는 매우 빠르고, 안전하며 그리고 AWS, Github Pages, Heroku, Netlify와 다른 호스팅 제공자와 같이 모든 곳에 배포될 수 있습니다.

더 알고 싶거나 기여를 하려면 [GitHub](https://github.com/gohugoio)를 확인하세요.
