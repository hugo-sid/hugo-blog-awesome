## 목차

이 테마는 블로그 포스트에 목차(ToC) 표시를 지원합니다.

## 파라미터

목차를 두가지 파라미터로 제어할 수 있습니다.
- 전역 `toc` 파라미터;
- 포스트 `toc` 파라미터.

포스트 `toc` 파라미터는 전역 `toc` 파라미터보다 더 높은 우선순위를 가집니다.

## 모든 포스트에 목차 활성화

모든 포스트(전역적으로) 목차를 활성화하려면 `hugo.toml`에서 `toc` 파라미터를 `true`로 설정하세요.

```toml
[params]
  toc = true
```

목차를 전역적으로 비활성화하려면, `toc` 파라미터를 무시하거나 `false`로 설정하세요.
 
## 특정 포스트에 목차 활성화

특정 포스트에 목차를 활성화하려면 포스트 설정에서 `toc` 파라미터를 `true`로 설정하세요.

    ```yaml
    ---
    title: 목차를 활성화하는 방법
    date: 2023-05-02
    toc: true
    ---
    ```

## 특정 포스트에 목차 비활성화

특정 포스트에서 목차를 비활성화하려면, 다음 두 단계를 거쳐야합니다

참고: 포스트의 `.Params.toc`은 `.Site.Params.toc`을 덮어쓸 것입니다. 다음 과정을 거치면, 포스트의 `toc` 파라미터는 `false`가 될 것입니다.

1.  `hugo.toml`에서 `toc` 파라미터를 `true`로 설정하세요.

    ```toml
    [params]
      toc = true
    ```

2.  목차를 비활성화하고자 하는 포스트의 머리말에 `toc = false`를 추가하세요

    ```yaml
    ---
    title: 목차를 활성화하는 방법
    date: 2023-05-02
    toc: false
    ---
    ```

## 목차 열기

기본적으로, 목차는 접혀있습니다.. 기본적으로 열리게 하려면, `hugo.toml`에서 `tocOpen` 파리미터를 `true`로 설정하세요.

```toml
[params]
  tocOpen = true
```

또는 단순히 `tocOpen` 파라미터를 포스트의 머리말에 추가하세요.

```yaml
---
title: 목차를 활성화하는 방법
date: 2023-05-02
tocOpen: true
---
```
